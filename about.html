Ecualizacion exponencial

-> Ajuste de contraste <-

Esta técnica se basa en modificar los valores de intensidad de los píxeles de la imagen para seguir una distribución exponencial, que puede ayudar a resaltar detalles oscuros o mejorar el contraste general.

Después de aplicar la transformación exponencial, el histograma resultante generalmente mostrará una mayor distribución de los valores de píxeles oscuros hacia rangos de intensidad media y alta, pero no necesariamente seguirá una distribución exponencial.

-> Formula <-

  • s = es el valor de intensidad del píxel de salida.
  • r = es el valor de intensidad del píxel de entrada.
  • c = es una constante que se ajusta para escalar el resultado al rango deseado de valores de píxeles (por ejemplo, [0, 255] para imágenes de 8 bits).
  • λ = es un parámetro que controla el indice de curvatura de la exponencial.
